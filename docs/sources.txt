====== primary_wire: Managing Sources ======

This guide covers how to add new sources to ''sources/sources.yaml''.

===== Source fields =====

^ Field    ^ Required ^ Description                                          ^
| slug     | Yes      | Short unique identifier used in CSV data files       |
| name     | Yes      | Full display name of the organization                |
| ticker   | No       | Stock ticker symbol — omit for non-public sources    |
| ir_url   | Yes      | Investor relations or official newsroom page URL     |
| notes    | No       | Useful context about finding press releases          |

===== Adding a new source =====

  - Check that the organization is not already in ''sources/sources.yaml''
  - Find the official IR or newsroom page URL
  - Choose a short, lowercase slug (e.g. ''apple'', ''jnj'', ''fed'')
  - Add an entry to ''sources/sources.yaml'':

<code yaml>
  - slug: apple
    name: Apple Inc.
    ticker: AAPL
    ir_url: https://investor.apple.com/news/default.aspx
    notes: null
</code>

  - Submit a pull request with the updated ''sources.yaml''

===== Slug naming conventions =====

  * Lowercase only
  * No spaces — use hyphens if needed (e.g. ''bank-of-america'')
  * Keep it short and obvious
  * Use the company's common name, not legal name (e.g. ''jnj'' not ''johnson-and-johnson'')

===== Non-company sources =====

Government agencies and other non-public organizations should omit the ''ticker'' field:

<code yaml>
  - slug: fed
    name: U.S. Federal Reserve
    ir_url: https://www.federalreserve.gov/newsevents/pressreleases.htm
    notes: Covers monetary policy decisions, FOMC statements, and regulatory actions.
</code>

===== Why the counts in README.txt don't match len(sources.yaml) =====

README.txt reports coverage as "N companies out of the ~500 in the S&P 500."
That number comes from ''missing_tickers.py'', which counts only tickers that
are *currently* constituents of the S&P 500. It will not match a simple count
of entries in ''sources/sources.yaml'', for two reasons:

  - Entries without a ''ticker'' (government agencies, other non-public
    sources) are not companies and are excluded from the count entirely.
  - Some tickers in ''sources.yaml'' may not currently be in the S&P 500 —
    e.g. the company was removed from the index after being added here, or it
    was added because it's of interest despite never being a constituent.
    These are still valid sources, just not counted toward the S&P 500
    coverage figure.

So ''sources.yaml'' entries = S&P 500 companies counted in README + non-S&P
tickers + non-ticker sources. Run ''python src/reporting/missing_tickers.py''
for the current authoritative count.

===== Disabling a source =====

Do not delete sources that have existing data. Instead add a note:

<code yaml>
  - slug: example_corp
    name: Example Corp
    ticker: EXMP
    ir_url: https://investor.example.com
    notes: Acquired by Acme Corp on 2026-06-01. No longer issuing independent press releases.
</code>
